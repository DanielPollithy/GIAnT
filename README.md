## npm commands

 - ```npm run coverage``` creates ./docs/coverage files
 - ```npm run docs``` creates ./docs/api files for ./src/*.js
 - ```npm run test``` runs all tests from ./test/*.js
 - ```npm run server``` runs the src/server.js
